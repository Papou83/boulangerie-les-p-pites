# Case Study : Les Pépites - Boulangerie Artisanale & Solidaire

## 🎯 Objectif du Projet
Créer une identité visuelle et une présentation interactive pour une boulangerie moderne. 
Le défi était de mêler l'aspect chaleureux de l'artisanat avec une communication percutante pour une cause caritative ("Cookies for a Cause").

## 🎨 Choix Graphiques (UI Design)
- **Palette :** Tons crème et marron pour l'authenticité, Rouge vif (#E63946) pour les appels à l'action solidaires.
- **Typographie :** Mélange de polices sans-serif modernes et de scripts artisanaux.
- **UX :** Parcours client fluide mettant en avant le produit avant la cause sociale.

## 🛠 Outils utilisés
- **Design :** Canva (Maquettes haute-fidélité)
- **Interaction :** Prototype interactif avec défilement automatique pour affichage en magasin.
- **Stratégie :** Copywriting français/anglais optimisé pour l'engagement.

## 🚀 Vision Webdesign
Ce projet démontre ma capacité à concevoir une interface cohérente, responsive et orientée vers la conversion, tout en gérant l'aspect émotionnel d'une marque.
